# IntelliJ

## Meta

```sh
CTRL+Shift+A - Find Action
CTRL + E - see recent files
```

## Windows

```sh
CTRL+CMD+F - Toggle fullscreen mode

FN+CMD+F4 - close tab

CTRL+TAB - Switch between tool window and editor window
CTRL+TAB Then 0 - Switch to SBT Window

SHIFT+CTRL+S - split windows
SHIFT+CTRL+U - unsplit windows
ALT+TAB - navigate between split windows
```

## Editor

```sh
SHIFT+CTRL+S - split windows
SHIFT+CTRL+U - unsplit windows
ALT+TAB - navigate between split windows
```

## Navigate

```sh

CTRL+H - find in path

CMD+N - Open class
SHIFT+CMD+N - Open file (also / for directories and file:lineno)
SHIFT+ALT+CMD+N - Open symbol  (also NameSpace.symbol)
```

## Code

```sh

CMD+P - show method parameters!!
CTRL+SPACE - Code complete [then press TAB to replace!]
SHIFT+CMD+ENTER - Finish formatting - brackets, semicolons

SHIFT+FN+F6 - Refactor

ALT+SHIFT+UP/DOWN - Structural selection

ALT+CMD+I - Auto-indent

F3 - Goto definition
CTRL+F3 - See structure
SHIFT+F3 - Goto implementation 

```

## Run/Debug

```sh
FN+CMD+F1 - Show error

ALT+SHIFT+D - Debug
SHIFT+FN+F10 - Run
```
